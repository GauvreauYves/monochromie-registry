# PDFs

All PDF files in this folder conform to **PDF/A-2b**
(ISO 19005-2:2011 — *Document management — Electronic document file format
for long-term preservation — Part 2: Use of ISO 32000-1*, conformance level B).

PDF/A-2b guarantees that the visual appearance of the document is fully
self-contained and reproducible without external resources. Fonts are
embedded, colour spaces are device-independent, and encryption is
prohibited. A conforming reader can render the document identically
decades from now without access to the original authoring environment.

Files are validated against ISO 19005-2:2011 at creation time.
